## ⚡ 3단계로 시작하기

### 1️⃣ Python 설치 (처음 한 번만)

Python이 없다면 설치하세요:
- **Windows/Mac**: https://www.python.org/downloads/ 에서 다운로드
- 설치 시 "Add Python to PATH" 체크 필수!

<br>

### 2️⃣ 파일 다운로드

이 저장소의 모든 파일을 다운로드하세요.

> 💡 **방법**: 초록색 "Code" 버튼 → "Download ZIP" 클릭

<br>

### 3️⃣ 실행

**Windows**: 폴더에서 `cmd` 입력 후 엔터

**Mac**: 폴더에서 우클릭 → "폴더에서 새로운 터미널"

그리고 아래 명령어를 **순서대로** 실행:

```bash
pip install requests
```

```bash
python fetch_regions.py
```

```bash
python run_search.py
```

## ❓ 문제 해결

### "python을 찾을 수 없습니다" 오류
→ Python 설치 시 "Add Python to PATH" 체크했는지 확인  
→ 컴퓨터 재부팅 후 다시 시도

### "pip를 찾을 수 없습니다" 오류
→ `python -m pip install requests` 시도

### "ModuleNotFoundError: No module named 'requests'" 오류
→ `pip install requests` 다시 실행

### 재고가 0대로만 나와요
→ 실제로 재고가 없을 수 있어요  
→ 다른 모델도 시도해보세요

## 💡 추가 팁

### 📍 특정 지역만 검색하고 싶다면:
```bash
python search_by_region.py
```
시도 → 시군구 → 모델 순서로 선택!

### 🔄 자동으로 계속 검색하고 싶다면:
실행 후 모드 선택에서:
- `2` = 5분마다 자동 검색
- `3` = 10분마다 자동 검색
- `4` = 30분마다 자동 검색

중단은 `Ctrl + C`
